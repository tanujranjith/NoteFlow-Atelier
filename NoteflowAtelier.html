Why you need an API key

NoteFlow is a local-first, privacy-focused app. We don't run AI servers or collect your data. To use the chat assistant, you need a free API key from Groq (a fast AI inference provider).

How to get your free Groq API key:

  1. Go to console.groq.com
  2. Sign up for a free account
  3. Navigate to API Keys and create a new key
  4. Copy and paste your key into the field below

Your API key is stored only in your browser (localStorage) and sent directly to Groq. We never see or store it.
