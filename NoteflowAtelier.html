Assistant

Flow Assistant panel behaviour and suggestions.

Panel
Panel defaultWhether the assistant opens with the workspace.
Context depthHow much workspace context Flow can see. More context means richer answers but larger token use.
Chat memory Stateless: Each message is treated independently. Faster, cleaner, and more private, but Flowy will not remember earlier chat context. Stateful: Flowy includes recent chat history so replies can follow the conversation. More contextual, but uses more tokens.
Recent messages Only used in Stateful mode. Include the last few chat messages with each request.
Confirmation modeHow Flow applies actions. High-risk actions (replace selection, bulk import, large batches) always require confirmation regardless of this setting.
Activity logEvery applied Flow action is recorded locally with an undo option. The log travels with .atelier / JSON backups.
